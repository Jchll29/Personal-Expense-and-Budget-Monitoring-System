# Personal Expense and Budget Monitoring System

A web-based application that enables users to manage their personal finances by recording income, tracking expenses, creating budgets, and generating financial reports.

## Features

- User Registration
- User Login
- Dashboard
- Income Management
- Expense Management
- Budget Management
- Reports & Analytics
- Notifications
- Profile Management
- Admin Panel

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Bootstrap 5
- Chart.js

---

# ▶️ How to Run the Project Using Visual Studio Code

## Step 1: Download or Clone the Repository

Clone the project using Git:

```bash
git clone https://github.com/yourusername/Personal-Expense-and-Budget-Monitoring-System.git
```

Or download the ZIP file from GitHub and extract it.

---

## Step 2: Open the Project in Visual Studio Code

1. Open **Visual Studio Code**.
2. Click **File → Open Folder**.
3. Select the **Personal-Expense-and-Budget-Monitoring-System** folder.
4. Click **Select Folder**.

---

## Step 3: Install Live Server

1. Click the **Extensions** icon (`Ctrl + Shift + X`).
2. Search for:

```
Live Server
```

3. Install **Live Server** by **Ritwick Dey**.

---

## Step 4: Run the Application

1. Open **index.html**.
2. Right-click anywhere inside the editor.
3. Select:

```
Open with Live Server
```

Or click the **Go Live** button in the bottom-right corner of Visual Studio Code.

The application will automatically open in your default web browser.

Example:

```
http://127.0.0.1:5500/index.html
```

or

```
http://localhost:5500/index.html
```
