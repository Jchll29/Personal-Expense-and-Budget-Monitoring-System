# Personal Expense and Budget Monitoring System

A web-based application that enables users to manage their personal finances by recording income, tracking expenses, creating budgets, and generating financial reports.

## Features

- User Registration
- User Login
- Dashboard
- Income Management
- Expense Management
- Budget Management
- Reports & Analytics
- Notifications
- Profile Management
- Admin Panel

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Bootstrap 5
- Chart.js

## Installation

1. Clone the repository

git clone https://github.com/username/Personal-Expense-and-Budget-Monitoring-System.git

2. Import database

expense_db.sql

3. Configure

config/database.php

4. Run using XAMPP

localhost/Personal-Expense-and-Budget-Monitoring-System
